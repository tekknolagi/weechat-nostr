# weechat-nostr

## Getting started

1. Install weechat
1. Clone this repo
1. Change directory into this repo's source tree
1. `virtualenv env`
1. `. ./env/bin/activate`
1. `pip install -r requirements.txt`
1. `weechat --run-command '/script load nostr.py'`
    (or run `weechat` and then `/script load nostr.py`)
1. Enjoy
